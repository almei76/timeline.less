Timeline.less

Timeline.less is a less based version of Timeline.css by Christian Fei with some php ressources and tricks.

Timeline.less is mobile-first and works great with the third version of Twitter Bootstrap. You can see it in action on my portfolio.

This code is not universal ! You have to set-up .item width in the timeline.less file to make it work with your respnsive layout !
The current configuration work with a custom Boostrap 3 grid (15px padding on the .container after the tablet media query).

If you find a way wich work with percents, you can make a pull-request or open an issue on Github.

Markup

<ul class="timeline">
	<!-- Begin a year -->
	<li class="pre-year"> </li>
	<li class="year">2013</li>
	<!-- Default : on the right -->
	<li class="item">
		<div class="item-container">
			<div class="item-content">
				<h1>Text exemple</h1>
				[Your content goes here]
			</div>
		</div>
	</li>
	<!-- On the left -->
	<li class="item item-left">
		[Same markup]
	</li>
	<!-- Featured (Full width) -->
	<li class="item-featured">
		[Same markup]
	</li>
	<!-- Important : End the Timeline -->
	<li class="timeline-end"> </li>
</ul>

  • On left

    Add the .item-left class to your .item element to position your box on the left of the timeline

  • Featured item

    Replace the .item class of your element by .item-featured to give it a full width.

    Warning !

    You next item must be a .item element without the .item-left class. See the PHP code bellow for further explaination.

  • Pictures

    You can add some pictures in your timeline. Be sure that they are responsive first !

  • New year

    Add a <li class="pre-year"> </li> (with the space) before every li.year to add margins between the end of the last year and the new year.

  • Timeline End

    The last li element must follow this markup to draw the timeline background : <li class="timeline-end"> </li>
